# FOMO Holders Frontend

React + Vite dashboard for checking FOMO wallet concentration on Solana tokens.

What it does:
- Lets you paste a token CA and run a scan.
- Shows FOMO % of total supply and top-holder bucket.
- Displays index/query stats from the FOMO Holders API.

This is the UI layer for `fomoholdersapi`.

API repo: https://github.com/xfajs/fomoholdersapi
